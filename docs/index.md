## 🛴快速开始

```sh
#安装git nodejs yarn

# clone the project
git clone https://github.com/windysoul-code/wuliblog.git

# enter the project directory
cd wuliblog

# install dependency 注意：如安装不成功请关闭淘宝源。
yarn install

# develop
yarn dev
```
