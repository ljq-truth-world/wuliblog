# 💚初始愿景

不是索要而是创造

此项目开始于2022年，意图在教学实践中，让受众从被动接受者过渡为实践创造者！期望以习题为承载，让未来的软件领域领导者了解

  • 通识内容：物理（物质世界演变的自然语言，普适的面向对象逻辑）

  • 专业技能：知识管理工具，版本管理工具

👨‍🦽说人话版：物理不能挂，所以还是要学。 学物理的同时顺便学学Markdown、咋写知识笔记、git版本控制啥的，感觉离大厂又近来。

#安装git nodejs yarn

# clone the project

# enter the project directory

# develop
yarn dev
